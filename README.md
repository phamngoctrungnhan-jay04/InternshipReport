## Overview
- **Role:** Cloud & Backend Engineer Intern
- **Timeline:** Sep 2025 – Jan 2026

- **Core Focus:** Cloud Infrastructure Design, Enterprise Security, and Backend Microservices

---

## Featured Project: ApexEV Platform
During the OJT program, I served as the Lead Architect and Backend Developer for a multi-role SaaS Electric Vehicle (EV) garage management system.

### Key Technical Contributions:
- **Cloud Architecture:** Designed a highly available AWS network topology utilizing VPC Subnets (Public/Private segmentation), Route 53, and Application Load Balancers (ALB) to achieve 100% core resource isolation.
- **Enterprise Security:** Implemented security controls using AWS Secrets Manager, IAM Least Privilege policies, and Spring Security + JWT to eliminate hardcoded credentials.
- **AI Integration:** Integrated an AI Chatbot leveraging AWS Bedrock (Claude 3.5 Sonnet) decoupled through AWS Lambda and API Gateway for zero backend compute overhead.
- **Containerization & DevOps:** Managed containerized services using AWS ECS Fargate and ECR, establishing automated system monitoring via Amazon CloudWatch.
